# 📊 Healthcare Data Analysis & Dashboard (Excel)

## 📌 Project Overview

This project focuses on analyzing healthcare data to extract meaningful insights related to patient trends, treatment patterns, and hospital performance using Microsoft Excel.

---

## 🎯 Objectives

* Analyze patient inflow and treatment distribution
* Identify high-performing departments
* Improve data visibility using dashboards
* Support data-driven decision-making

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers

---

## 📂 Files Included

* `data/raw_healthcare_data.xlsx` → Original dataset used for analysis
* `dashboard/healthcare_dashboard.xlsx` → Cleaned dataset with dashboard and insights

---

## 🧹 Data Cleaning

* Handled missing values
* Structured raw data for analysis
* Ensured consistency in columns and formats

---

## 📊 Key Analysis Performed

* Patient inflow trends over time
* Department-wise performance analysis
* Treatment distribution insights
* Data filtering using slicers

---

## 📈 Dashboard Features

* Interactive filters using slicers
* Visual representation using charts
* Easy-to-understand summary of key metrics


---

## 🔍 Key Insights

* Identified peak periods of patient inflow
* Certain departments handle the majority of cases
* Data visualization improved clarity and reporting

---

## 🚀 Conclusion

This project demonstrates how Excel can be used for end-to-end data analysis, from data cleaning to dashboard creation, helping healthcare organizations make better decisions.
